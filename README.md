# install
How to Install my Templates

> [!IMPORTANT]
> Before installing my templates, let me know in advance if you intend to use my templates on a commerical use blog.
> [CLICK HERE](https://warrenwoodhouse.blogspot.com/contact) to get in touch.
> My templates are free for personal use only and for commerical use upon request via the contact form.

1. Download my template in the form of the HTML File
2. Log into Blogger by [CLICKING HERE](https://www.blogger.com/)
3. Select the blog you would like the template to be used on
4. Head to Themes then select the drop down arrow next to the word CUSTOMIZE
5. In CUSTOMIZE, select Restore and then upload the HTML File (one of my templates you downloaded earlier)
